问题

通过 yarn 包管理器安装 yarn install

报错：

原因：

​ 开始以为是下载源的问题，但是切换到淘宝源后依然无法解决问题，还是报这个问题。并且自己通过 npm 包管理器安装时，也会报同样的错误，各种查找了一圈，发现是【HTTPS 证书验证失败】导致的。

解决方案：

​ 将 yarn 配置中的 strict-ssl 设置为 flase , 在 yarn config 信息中， strict-ssl 为 true，表示需要验证 HTTPS 证书。我们可以将 strict-ssl 设置为 false，跳过 HTTPS 证书验证。
