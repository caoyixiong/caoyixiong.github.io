ISO 639-1 Language Code 备忘清单

这是一个符合 ISO 639-1 标准的 ISO 语言代码列表，它为多语言网站提供参考。

说明

介绍

  • ISO 639-1 是用于对语言进行分类的标准化命名法
  • ISO 639-1 代码涵盖世界主要语言
  • 这些代码是用于指示语言的有用的国际和正式速记
  • 两个字母的代码
  • 用于多语言网站，即国际化 (i18n)
  • i18n 语言代码
